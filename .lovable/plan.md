

# Pharaoh B. 1111 - Egyptian-Themed Portfolio Website

## Overview
A luxurious, mystical single-page website with a black background, golden Egyptian-inspired typography, and an interactive 3D golden torus that floats in the background and responds to scrolling. The design evokes ancient Egyptian royalty while maintaining modern elegance.

---

## Visual Design System

### Color Palette
- **Background**: Deep black (#0A0A0A)
- **Primary Gold**: Rich pharaoh gold (#D4AF37)
- **Accent Gold**: Warm antique gold (#B8860B)  
- **Text**: Cream white with golden highlights
- **Subtle accents**: Deep emerald and bronze

### Typography
- **Headings**: Serif Egyptian-style font (Cinzel or Playfair Display) with golden gradient effects
- **Body**: Clean modern font for readability
- **Decorative elements**: Hieroglyphic-inspired ornaments

---

## 3D Golden Torus Effect
- A slowly rotating, shimmering 3D torus using React Three Fiber
- Positioned as a subtle background element with parallax scroll effect
- Golden metallic material with soft glow/bloom effect
- Opacity and position shift as user scrolls through sections
- Creates an "artifact floating in sacred space" atmosphere

---

## Site Sections

### 1. Hero Section
- Full-screen black background with the golden torus centerpiece
- "Welcome to the Echo of Your Own Becoming" headline with Egyptian serif font
- "Pharaoh B. | Writer. Architect. Mirror. Messenger" tagline
- "Enter the Canon" call-to-action button with golden border

### 2. Navigation
- Fixed minimal navbar with golden text
- "PHARAOH B. 1111" logo/brand mark on left
- Navigation links: About, The Canon, Echoverse, Shop
- Location indicator: "Houston, Texas"

### 3. About Section
- Split layout with text and decorative imagery placeholder
- Bio/story content with elegant typography
- Golden divider ornaments

### 4. The Canon Section
- Showcase for writings and publications
- Book/product cards with hover effects
- Golden frame styling around featured items

### 5. Echoverse Section
- Creative universe exploration area
- Grid or carousel of projects/concepts
- Mystical imagery placeholders

### 6. Shop Section (Frontend Only)
- Product display grid
- Product cards with images, titles, prices
- "Coming Soon" or placeholder checkout experience
- No backend functionality - just visual layout

### 7. Footer
- Newsletter signup form (frontend only - styled input)
- Contact form (frontend only - styled layout)
- Social media links
- Copyright and Egyptian-inspired decorative border

---

## Interactive Features
- Smooth scroll navigation between sections
- Torus parallax responds to scroll position
- Hover animations on buttons and cards
- Golden shimmer effects on interactive elements
- Fade-in animations as sections enter viewport

---

## Technical Approach
- React Three Fiber for 3D torus with metallic gold material
- Scroll-based animations using intersection observers
- Custom Egyptian-inspired UI components
- Fully responsive design
- All forms are visual placeholders (backend to be added later)
